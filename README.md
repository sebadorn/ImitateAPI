# Imitate API


## Requirements

* openssl -- but only if SSL certificates for localhost have to be generated
* python3 -- version 3.5 or higher, tested with 3.6.9


## Install

Using `pip3`:

```bash
pip3 install imitate-api
```

or

```bash
python3 -m pip install imitate-api
```
